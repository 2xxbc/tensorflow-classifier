# tensorflow-classifier
TensorFlow image classification demo using Fashion-MNIST with training, validation, evaluation, and model saving.

# TensorFlow Classifier (Fashion-MNIST)

This project trains a simple TensorFlow/Keras neural network to classify Fashion-MNIST images.

## Requirements
- Python 3.11+
- Windows / macOS / Linux

## Setup (Windows PowerShell)
```powershell
cd C:\Users\your folder\TensorFlow
py -3.11 -m venv .venv
. .\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install tensorflow
